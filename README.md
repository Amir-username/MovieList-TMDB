
# Movies App

A movies app with [TMDB API](https://www.themoviedb.org/) and React-Ts/Tailwindcss

## Demo

https://movie-list-tmdb.vercel.app/


## Run Locally

Clone the project

```bash
  git clone https://Amir-username/MovieList-TMDB
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
